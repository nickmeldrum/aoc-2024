# aoc-2024

[aoc website: https://adventofcode.com/2024](https://adventofcode.com/2024)

## install

```
yarn
```

## setup new day:

(where n = day number)
will:
 1. make dir: `dayn`
 2. make input file: `input` and get contents from input url:
 3. make template input.mjs

```
yarn setup n
```

## run

```
yarn day1
```
